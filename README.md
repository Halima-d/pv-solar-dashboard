# PV Solar Dashboard

**Predict photovoltaic (PV) energy production using Python and Streamlit.**  
An interactive dashboard that helps users explore and predict solar panel energy output based on historical and environmental data.

---

## Features
- Real-time PV energy prediction with machine learning models.
- Interactive **Streamlit dashboard** for easy input and visualization.
- Visual graphs to analyze solar energy trends.
- User-friendly interface suitable for students, engineers, and researchers.

---

## Technologies
- Python
- Streamlit
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
